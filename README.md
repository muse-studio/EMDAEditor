#  **EMDAツール**

---

## 📄 概要
Tkinter を用いて開発した、木構造制作ツールです。

複数の垂直線をドラッグして、他の線にスナップ（吸着）させることで**木構造**を形成できます。

また、ズーム・Undo/Redo・保存/読み込みに対応し、グループ分割やテキスト入力などのUI機能も備えています。 

---

## 🚀 実行方法

1.  Pythonをインストール
   
2. 添付されているコード(試作1)をダウンロード  
   
3. ターミナルまたはコマンドプロンプトで実行

4. ウィンドウが開き、初期状態で6本の線が表示されます。

---
## 🎮 基本操作方法

線の上端をドラッグすれば、自由に動かせます。

他の線に近づけるとその線にくっつきます。

くっついた線は、くっつけた線の動きに追従します。

---
##  機能一覧

| 分類              | 操作内容            | 操作方法                     |
| --------------- | --------------- | ------------------------ |
| **線の操作**        | 線の上端をドラッグして移動   | マウス左ドラッグ                 |
|                 | 線を他の線にスナップ（接続）  | 上端を他線に近づける               |
|                 | スナップされた線を親に追従   | 親線を動かすと自動追従              |
| **追加モード**       | 追加モードを有効化 / 無効化 | 「＋ 追加モード ON/OFF」ボタン      |
|                 | 新しい線を追加         | 赤い丸（上端ガイド）をクリック          |
| **削除モード**       | 削除モードを有効化 / 無効化 | 「− 削除モード ON/OFF」ボタン      |
|                 | 線を削除            | 青い丸（上端）をクリック             |
| **分割モード**       | 分割モードを有効化 / 無効化 | 「｜ 分割モード ON/OFF」ボタン      |
|                 | グループを分割 / 結合    | 下段のグループ行をクリック            |
| **ズーム・水平移動**      | 拡大 / 縮小         | マウスホイール or `+ / -` キー    |
|                 | 水平移動            | Shift + ホイール or ← / → キー |
|                 | 表示範囲リセット        | `0` キー or 「リセット」メニュー     |
| **ファイル**      | 状態を保存           | メニュー → ファイル → 保存         |
|                 | 状態を読み込み         | メニュー → ファイル → 読み込み       |
| **Undo / Redo** | 操作を戻す           | `Ctrl + Z`               |
|                 | 操作をやり直す         | `Ctrl + Y`               |
| **表示範囲設定**        | 表示範囲を直接入力       | 「表示範囲(％)」欄に数値を入力し「適用」    |
| **線の本数設定**      | 線の本数を変更         | 「線の本数」欄で数値を指定し「適用」       |
| **グループ行**       | グループテキストを編集     | 下段の入力欄に文字を入力             |
