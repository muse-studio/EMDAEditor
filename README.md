#  **EMDAツール**

---

## 📄 概要
Tkinter を用いて開発した、木構造制作ツールです。
複数の垂直線をドラッグして、他の線にスナップ（吸着）させることで**木構造**を形成できます。
また、ズーム・Undo/Redo・保存/読み込みに対応し、グループ分割やテキスト入力などのUI機能も備えています。 

---

##  機能一覧

| 操作方法 | 内容 |
| :--- | :--- |
| マウスホイール | ズームイン・アウト |
| ←, → キー | 表示範囲の左右移動 |
| 0 キー | 表示範囲のリセット |
| +, - キー | キーボードでのズームイン・アウト |


| ボタン            | 説明                             |
| -------------- | ------------------------------ |
| **＋ 追加モード ON** | 赤い丸（上端ガイド）をクリックすると、新しい線を挿入します。 |
| **− 削除モード ON** | 青い丸（上端）をクリックすると、対象の線を削除します。    |
| **｜ 分割モード ON** | 下段のグループ行をクリックして、分割線を追加／削除できます。 |


| 項目               | 内容                                |
| ---------------- | --------------------------------- |
| **線の本数**         | 上部の「線の本数」欄で指定可能。数値を入力して「適用」をクリック。 |
| **表示範囲**         | 「表示範囲(％)」を変更してズーム範囲を指定。           |
| **Undo/Redo 履歴** | 線やグループ構造の状態を保存し、いつでも戻せます。         |


| 機能                | 説明                               |
| ----------------- | -------------------------------- |
| **保存**            | 現在の線構造とテキスト情報を `.json` 形式で保存します。 |
| **読み込み**          | 保存した `.json` ファイルを復元します。         |


---

## 🚀 実行方法

1.  Pythonをインストール
   
2. 添付されているコードをダウンロード  
   
3. ターミナルまたはコマンドプロンプトで実行

4. ウィンドウが開き、初期状態で6本の線が表示されます。
