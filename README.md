# EMDAtool
EMDA簡略化の為のツール作成

Tkinter を用いて開発した、木構造制作ツールです。
複数の垂直線をドラッグして、他の線にスナップ（吸着）させることで**木構造**を形成できます。
また、ズーム・Undo/Redo・保存/読み込みに対応し、グループ分割やラベル入力などのUI機能も備えています。

🧩 主な機能
基本操作

垂直線をマウスでドラッグして移動できます。

線の上端を他の線に近づけるとスナップ（接続）します。

スナップされた線は、親線を動かすと連動して動きます。

追加・削除モード

＋ 追加モード ON：赤い丸（上端ガイド）をクリックすると新しい線を挿入します。

− 削除モード ON：青い丸（上端）をクリックすると線を削除します。

分割モード

｜ 分割モード ON：下段のグループ行をクリックして分割線を追加・削除できます。

グループごとにテキスト入力欄を設定可能です。

ズーム・パン操作
操作方法	内容
マウスホイール	ズームイン・アウト
Shift + ホイール	横方向のパン移動
中クリック／右ドラッグ	パン移動
+, - キー	キーボードでズーム
←, → キー	キーボードでパン
0 キー	表示範囲をリセット (0〜100%)
💾 ファイル操作

保存：現在の線構造とテキスト情報を .json 形式で保存します。

読み込み：保存した .json ファイルを復元します。

Undo / Redo：Ctrl+Z / Ctrl+Y で状態を戻す／やり直すことができます。

⚙️ 設定項目
項目	内容
線の本数	上部の「線の本数」欄で指定可能。数値を入力し「適用」で更新。
表示範囲	「表示範囲(％)」を指定してズーム範囲を変更可能。
Undo/Redo 履歴	線やグループ構造の変更を記録し、いつでも戻せます。

🚀 実行方法

Pythonをインストール

このリポジトリをクローンまたはダウンロード

ターミナルまたはコマンドプロンプトで実行：

python 試作1.py


ウィンドウが開き、初期状態で6本の線が表示されます。
